## 项目说明
本项目使用 **Vue**框架进行开发,服务端使用**SpringBoot + MyBatis**来实现,数据库采用了**MySQL**


本项目的客户端和管理端使用 **Vue** 框架来实现，服务端使用 **Spring Boot + MyBatis** 来实现，数据库使用了 **MySQL**。

## 项目功能

- 用户登录注册
- 登录验证码
- SpringEmail邮箱
- 定时任务
- 每日任务清单（每天凌晨12点自动清空）
- 用户各平台账号密码进行存储记录
- 模糊查询

## 技术栈

### 后端

**SpringBoot + MyBatis + Redis**

### 前端

**Vue3.0 + JavaScript+ Vue-Router + Vuex + Axios + ElementPlus**
